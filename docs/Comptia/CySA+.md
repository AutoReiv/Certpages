# CompTIA CySA+

## Overview

Exam Code: 
CS0-001

Prerequisites:
None

Recommended Experience: 
Some networking and security experience

Number of Questions:
85

Test Format:
Multiple choice and performance-based

Length of Test:
165 minutes

Passing Score:
750 (on a scale of 100-900)

Test Objectives:
[CS0-001 Objectives](https://comptiacdn.azureedge.net/webcontent/docs/default-source/exam-objectives/comptia-cysa-cs0-002-exam-objectives.pdf)

Cost:
$349

Registration:
[Register Here](https://www.comptia.org/certifications/cybersecurity-analyst#buyoptions)

Expiration:
3 years after the certification is acquired.

Renewal:
Get 60 or more credits of Continuing Education Units in 3 years
