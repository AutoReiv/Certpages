# AWS Certified DevOps Engineer - Professional

## Overview

Exam Code: 
DOP-C01

Prerequisites:
None

Recommended Experience: 
An AWS associate certification is recommended but not required.

Number of Questions:
80

Test Format:
Multiple choice, multiple answer

Length of Test:
170 minutes

Passing Score:
Around 75%
