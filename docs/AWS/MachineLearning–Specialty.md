# AWS Certified Machine Learning – Specialty

## Overview

Exam Code: 
MLS-C01

Prerequisites:
None

Recommended Experience: 
An AWS associate certification is recommended but not required.

Number of Questions:
65

Test Format:
Multiple choice, multiple answer

Length of Test:
170 minutes

Passing Score:
Around 70%

Test Objectives:
[MLS-C01 Objectives][1]
[1]: https://d1.awsstatic.com/training-and-certification/docs-ml/AWS%20Certified%20Machine%20Learning%20-%20Specialty_Exam%20Guide%20(1).pdf

Cost:
$300

Registration:
[Register Here][2]
[2]: https://www.aws.training/certification?rightcta=mlexam

Expiration:
3 years after the certification is acquired.

Renewal:
[Take the current specialty-level exam][3]
