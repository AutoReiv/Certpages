# AWS Certified Solutions Architect - Professional

## Overview

Exam Code: 
SAP-C01

Prerequisites:
None

Recommended Experience: 
An AWS associate certification is recommended but not required.

Number of Questions:
80

Test Format:
Multiple choice, multiple answer

Length of Test:
170 minutes

Passing Score:
Around 75%

Test Objectives:
[SAP-C01 Objectives][1]
[1]: https://d1.awsstatic.com/training-and-certification/docs-sa-pro/AWS_Certified_Solutions_Architect_Professional-Exam_Guide_EN_1.2.pdf
Cost:
$300

Registration:
[Register Here][2]
[2]: https://www.aws.training/certification?src=arc-pro

Expiration:
3 years after the certification is acquired.

Renewal:
[Take the current professional-level exam][3]
