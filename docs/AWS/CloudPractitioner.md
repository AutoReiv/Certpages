# AWS Certified Cloud Practitioner

## Overview

Exam Code: 
CLF-C01

Prerequisites:
None

Recommended Experience: 
None

Number of Questions:
65

Test Format:
Multiple choice, multiple answer

Length of Test:
90 minutes

Passing Score:
Around 70%

Test Objectives:
[CLF-C01 Objectives][1]
[1]: https://d1.awsstatic.com/training-and-certification/Docs%20-%20Cloud%20Practitioner/AWS_Certified_Cloud_Practitioner_Exam_Guide_v2.1.pdf

Cost:
$100

Registration:
[Register Here][2]
[2]: https://www.aws.training/certification?src=practitioner

Expiration:
3 years after the certification is acquired.

Renewal:
[Take the Cloud Practitioner exam OR earn and associate-level or professional-level certification][3]
[3]: https://aws.amazon.com/certification/recertification/
