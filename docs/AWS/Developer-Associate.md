# AWS Certified Developer – Associate

## Overview

Exam Codes: 
DVA-C01

Prerequisites:
None

Recommended Experience: 
Some experience with AWS is recommended 

Number of Questions:
60

Test Format:
Multiple choice, multiple answer

Length of Test:
130 minutes

Passing Score:
Around 70%

Test Objectives:
[DVA-C01 Objectives][1]
[1]: https://d1.awsstatic.com/training-and-certification/docs-dev-associate/AWS_Certified_Developer_Associate-Exam_Guide_EN_1.4.pdf

Cost:
$150

Registration:
[Register Here][2]
[2]: https://www.aws.training/certification?src=dev-assoc

Expiration:
3 years after the certification is acquired.

Renewal:
[Take the current Associate exam again][3] OR Earn a AWS professional level certification
[3]: https://aws.amazon.com/certification/recertification/
