# AWS Certified Advanced Networking - Specialty

## Overview

Exam Code: 
ANS-C00

Prerequisites:
None

Recommended Experience: 
An AWS associate certification is recommended but not required.

Number of Questions:
65

Test Format:
Multiple choice, multiple answer

Length of Test:
170 minutes

Passing Score:
Around 70%

Test Objectives:
[ANS-C00 Objectives][1]

Cost:
$300

Registration:
[Register Here][2]
[2]: https://www.aws.training/certification?src=an-spec

Expiration:
3 years after the certification is acquired.

Renewal:
[Take the current specialty-level exam][3]
