# Red Hat Certified System Administrator

## Overview

Exam Code: 
EX200

Prerequisites:
None

Recommended Experience: 
Some linux experience

Number of Questions:
Performance-based evaluation

Test Format:
Hands-on practical exam

Length of Test:
2.5 Hours

Passing Score:
Pass or Fail

Test Objectives:
[EX200 Objectives](https://www.redhat.com/en/files/resources/en-rhtr-ex200-rhel-6-exam-objectives-12244737.pdf)

Cost:
$400

Registration:
[Register Here](https://www.redhat.com/en/services/training/ex200-red-hat-certified-system-administrator-rhcsa-exam)

Expiration:
3 years after the certification is acquired.

Renewal:
Retake RHCSA(EX200), pass RHCE(EX300), or pass any RHCE while holding RHCE(EX300)
