    AWS Certified Solutions Architect - Associate¶

    Overview¶

    Exam Code: SAA-C01

    Prerequisites: None

    Recommended Experience: Some experience with AWS is recommended

    Number of Questions: 60

    Test Format: Multiple choice, multiple answer

    Length of Test: 130 minutes

    Passing Score: Around 70%

    Test Objectives: SAA-C01 Objectives

    Cost: $150

    Registration: Register Here

    Expiration: 3 years after the certification is acquired.

    Renewal: Take the current Associate exam again OR Earn a AWS professional level certification
