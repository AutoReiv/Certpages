    AWS Certified Security - Specialty¶

    Overview¶

    Exam Code: SCS-C01

    Prerequisites: None

    Recommended Experience: An AWS associate certification is recommended but not required.

    Number of Questions: 65

    Test Format: Multiple choice, multiple answer

    Length of Test: 170 minutes

    Passing Score: Around 70%

    Test Objectives: SCS-C01 Objectives

    Cost: $300

    Registration: Register Here

    Expiration: 3 years after the certification is acquired.

    Renewal: Take the current specialty-level exam
